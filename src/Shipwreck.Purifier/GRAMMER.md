# 代名詞
- 我々 → 私たち
- 僕 → 私

# 動詞

- 終止形。
- 命令系。 → 終止形+ぷり。

# 形容動詞

- です。 → ぷり。

# 助動詞

- だ。 → ぷり。

# 助詞

## 接続助詞

## 終助詞

- ぞ。 → ぷりね。
- や。 → 終止形+ぷりよ。
- わ。 → ぷり。
- こと。 → <s>ぷり。</s>体言止めは無視
- てよ。 → ぷりよ。
- ことよ。 → ぷりよ。
- もの。 → <s>ぷり。</s>体言止めは無視

## 間投助詞

- さ。 → ぷり。
- よ。 → ぷりよ。
- ぜ。 → ぷりよ。
- ね。 → ぷりね。
- な。 → ぷり。
- の。 → ぷり。
